3 · Install instructions

Easy path — Load unpacked (works 100%):
  1. Download the .zip above and unzip it to a folder.
  2. Open chrome://extensions
  3. Turn on Developer mode (top-right toggle).
  4. Click Load unpacked → select the unzipped proctor-extension folder.
  5. Come back here and click Re-check — status should flip to green.
The .crx path (to test self-hosted install friction):
  1. chrome://extensions, Developer mode on.
  2. Drag the downloaded .crx onto the page.
  3. Chrome will likely block it ("can only be added from the Chrome Web Store"). That friction is exactly why real install = Web Store (BYOD) or force-install policy (lab machines).

On install you'll see the warning "Read your browsing history" (the tabs permission) — that's what students would see.
