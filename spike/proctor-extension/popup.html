Proctor Extension Spike

What a proctoring extension can & cannot see — live proof.

✅ CAN see — every tab in every window

Click "Scan now".

✅ CAN also see (browser-wide)

  • Which tab is active, and every tab switch / new tab / closed tab (incl. the URL).
  • When focus leaves the browser entirely (alt-tab to another app).
  • Every navigation (URL changes) in any tab.
  • With a content script + permission: read/observe the contents of the HackerRank tab itself (paste, typing, fullscreen) — not shown here.

→ Open chrome://extensions → this extension → service worker (Inspect) and watch the console while you switch tabs / alt-tab.

⛔ CANNOT see (no extension API exists)

  • Other programs / the Windows Task Manager / Linux processes. No browser-extension API exposes the OS process list.
  • Apps outside the browser, a transparent overlay window (e.g. Cluely), a second device/phone, a VM.
  • Tabs in a different browser (e.g. a second Chrome profile, Firefox, Edge) — each browser is isolated.

→ Seeing OS processes/other apps requires a Native Messaging host: a separate native binary you install on the machine that the extension talks to. That's a real app install, not "just an extension."
