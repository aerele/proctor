A long wooden fence is made of L identical cells standing in a row, numbered 1 to L from left to right. At the start every cell is bare (unpainted).

A painter performs Q operations. In each operation she picks a contiguous stretch of the fence from cell l to cell r (inclusive) and paints every cell in that range. Ranges from different operations may overlap, touch, or be completely separate. Painting an already-painted cell simply leaves it painted (a cell does not get 'more painted').

After all Q operations are done, the painter wants to know how much of the fence has any paint on it at all - that is, how many cells have been painted at least once.

For example, with L = 10 and the operations [1, 3] and [2, 5]: the first paints cells 1, 2, 3 and the second paints cells 2, 3, 4, 5. Together the painted cells are 1, 2, 3, 4, 5 - so 5 cells are painted.

Given L and the Q operations, output the number of cells painted at least once.

Input Format

The first line contains two integers L and Q: the number of fence cells and the number of paint operations. Each of the next Q lines contains two integers l and r, describing one paint operation that covers cells l to r inclusive.

Constraints

1 <= L <= 100000 1 <= Q <= 100000 1 <= l <= r <= L

Output Format

A single integer: the number of fence cells painted at least once.

Sample Input 0

10 2
1 3
2 5

Sample Output 0

5

Explanation 0

Operation [1,3] paints cells 1,2,3. Operation [2,5] paints 2,3,4,5. Union is {1,2,3,4,5}, so 5 cells.
