Maya runs a small snack stall outside her college. She installed an old vending machine that returns change using coins of only four kinds: 25, 10, 5 and 1.

The machine has a quirk: whenever it returns change, it hands back as FEW coins as possible. For example, to return 41 it gives one 25, one 10, one 5 and one 1 - just 4 coins. It never does something silly like returning forty-one 1-coins.

Given an amount of change N, print the minimum number of coins (using denominations 25, 10, 5 and 1) that sum exactly to N.

Input Format

A single line containing one integer N, the amount of change to be returned.

Constraints

0 <= N <= 10000

Output Format

A single integer: the minimum number of coins that sum exactly to N.

Sample Input 0

41

Sample Output 0

4

Explanation 0

41 = 25 + 10 + 5 + 1, which uses 4 coins. No combination uses fewer coins.
