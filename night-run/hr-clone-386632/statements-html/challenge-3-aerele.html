In Numbergarh, bus tickets carry a 6-digit number (leading zeros are allowed, so 007123 is a valid ticket). The locals have a charming superstition: a ticket is LUCKY if the sum of its first three digits equals the sum of its last three digits.

For instance, 123222 is lucky because 1 + 2 + 3 = 6 and 2 + 2 + 2 = 6. The ticket 100001 is also lucky (1 + 0 + 0 = 1 and 0 + 0 + 1 = 1), but 123456 is not (1 + 2 + 3 = 6, but 4 + 5 + 6 = 15).

Aarav collected a bunch of tickets today and wants to know how many of them are lucky. Given M tickets, count how many are lucky.

Input Format

The first line contains an integer M, the number of tickets. Each of the next M lines contains a 6-character string representing a ticket number (digits only, may include leading zeros).

Constraints

1 <= M <= 100000 Each ticket is exactly 6 digits (0-9).

Output Format

A single integer: the number of lucky tickets.

Sample Input 0

3
123222
100001
999999

Sample Output 0

3

Explanation 0

All three tickets are lucky: 1+2+3=2+2+2=6; 1+0+0=0+0+1=1; 9+9+9=9+9+9=27.
