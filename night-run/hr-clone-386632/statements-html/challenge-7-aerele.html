Two friends, Anu and Bala, found a treasure chest containing N gold coins. The coins have different weights, and the friends want to divide ALL the coins between themselves so that each person carries away exactly the same total weight. No coin can be cut or left behind - every coin must go to exactly one of the two friends.

They want to know whether such a perfectly fair split is even possible.

For example, coins with weights [1, 5, 11, 5] can be split fairly: Anu takes {1, 5, 5} (total 11) and Bala takes {11} (total 11). But coins with weights [1, 2, 3, 5] cannot be split into two groups of equal weight, no matter how you try.

Given the weights of the N coins, determine whether they can be divided into two groups with equal total weight. Print YES if it is possible, otherwise print NO.

Input Format

The first line contains an integer N, the number of coins. The second line contains N space-separated integers, the weights of the coins.

Constraints

1 <= N <= 20 1 <= weight of each coin <= 100000

Output Format

Print YES if the coins can be split into two groups of equal total weight, otherwise print NO.

Sample Input 0

4
1 5 11 5

Sample Output 0

YES

Explanation 0

Split into {1, 5, 5} (sum 11) and {11} (sum 11). Equal totals, so the answer is YES.
